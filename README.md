#  Gestion de Notes - JavaFX

##  Description
Application desktop de gestion de notes développée en JavaFX avec base de données PostgreSQL.

##  Technologies utilisées
- Java 17
- JavaFX 21
- PostgreSQL
- Maven
- JDBC

##  Fonctionnalités
- Gestion des étudiants
- Gestion des matières
- Ajout / modification / suppression des notes
- Calcul automatique des moyennes
- Système d’audit des opérations
- Gestion des utilisateurs (ADMIN / ENSEIGNANT / ETUDIANT)

##  Installation

### 1. Cloner le projet
```bash
git clone https://github.com/TON_USER/gestion-notes-javafx.git
cd gestion-notes-javafx
2. Installer les dépendances
mvn clean install
3. Lancer l'application
mvn javafx:run
